A URL needs to written in letter, numbers or these symbols $ - _ or . any other symbols need to use a URL excape code such as %20 is space, %3f is a question mark and %2b is a slash. So file name need to use these escape code unless special character are not used or the URL is in quotation marks.OR you don't use spaces in file names.
